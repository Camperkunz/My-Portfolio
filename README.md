# Frontend Portfolio

A sleek, responsive, and high-performance portfolio website designed to showcase my creative works and technical skills. Built with a focus on clean code, smooth animations, and modern UI/UX principles.

## 🚀 Tech Stack

- **Core:** [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)

## ✨ Key Features

- **Responsive Design:** Optimized for all screen sizes from mobile to desktop.
- **Dynamic Projects Gallery:** Interactive display of my latest work.
- **Smooth Transitions:** Polished animations using Framer Motion.
- **Clean Architecture:** Component-based structure for easy maintenance.
